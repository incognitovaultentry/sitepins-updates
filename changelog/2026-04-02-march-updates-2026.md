In March, our updates we're based on feedbacks from teams actually using Sitepins on their daily workflow.

Here's everything we shipped last month:

- **Live User Indicators** — See live avatars in the editor showing who's currently active on the same file, with real-time updates and name tooltips on hover.
- **External Image URL Support** — Paste any external image URL (CDN, third-party host, etc.) directly into the rich text editor without needing to re-upload.
- **iFrame Preview in Rich Editor** — Embed and preview iFrames (like YouTube videos or charts) live inside the HTML block as you write.
- **Account Overview Refresh** — A cleaner dashboard showing your current plan, organization count, number of sites, users, and usage indicators at a glance.
